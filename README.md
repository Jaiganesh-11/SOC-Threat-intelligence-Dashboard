🛡️ SOC Threat Intelligence Dashboard
📌 Project Overview

The SOC Threat Intelligence Dashboard is an interactive Power BI dashboard designed to analyze cybersecurity incidents and transform raw security data into actionable business insights. The project demonstrates the complete analytics workflow, including data cleaning, data transformation, data modeling, DAX calculations, KPI development, and data visualization.

The dashboard enables security teams and decision-makers to monitor cyberattack trends, assess financial impacts, identify vulnerable industries, and evaluate attack sources through an intuitive and interactive reporting interface.

🎯 Project Objectives
Analyze cybersecurity incidents across multiple countries and industries.
Identify attack patterns and security vulnerabilities.
Monitor financial losses caused by cyber threats.
Evaluate affected users and attack frequency.
Support data-driven cybersecurity decision-making.
