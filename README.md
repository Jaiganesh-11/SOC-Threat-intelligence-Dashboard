# 🛡️ SOC Threat Intelligence Dashboard

## 📌 Project Overview

The **SOC Threat Intelligence Dashboard** is an interactive Power BI solution developed to analyze cybersecurity incidents and transform raw security data into actionable business intelligence. This project demonstrates an end-to-end analytics workflow, including **data cleaning, data transformation, data modeling, DAX calculations, KPI development, and interactive data visualization**.

The dashboard enables organizations to monitor cyberattack trends, assess financial impacts, identify vulnerable industries, evaluate security weaknesses, and support data-driven cybersecurity decision-making through comprehensive reporting and analysis.

---

## 🎯 Project Objectives

* Analyze cybersecurity incidents across multiple countries and industries.
* Identify attack patterns, sources, and security vulnerabilities.
* Measure the financial impact of cyberattacks.
* Evaluate affected users and attack frequency.
* Support proactive and data-driven security strategies.

## 📊 Dashboard Features

### Executive KPIs

* Total Financial Loss
* Total Affected Users
* Maximum Attack Duration
* Total Attack Count
* Industry Coverage

### Cyber Threat Analysis

* Financial Loss by Country
* Financial Loss by Attack Type
* Attack Distribution Analysis
* Security Vulnerability Assessment
* Attack Source Analysis

### Industry Risk Assessment

* Industry-wise Financial Impact
* Target Industry Comparison
* Industry-Specific Threat Exposure
* Multi-Year Cybersecurity Trends

### Trend Monitoring

* Year-over-Year Financial Loss Analysis
* User Impact Trends
* Attack Growth Monitoring
* Risk Pattern Identification

## 🔍 Key Insights

* Identified countries with the highest financial losses due to cyberattacks.
* Analyzed attack types contributing most significantly to organizational risk.
* Evaluated commonly exploited security vulnerabilities and attack vectors.
* Assessed the impact of cyber threats across multiple industries.
* Compared attack sources and their influence on affected users.
* Tracked cybersecurity trends and financial losses over time.
